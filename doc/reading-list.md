# Dark Software Factories — Reading List

This document collects resources on autonomous software factories, agentic coding, and the transition from AI-assisted to fully autonomous software development. Topics covered include the dark factory pattern, multi-agent orchestration, maturity models, and enterprise adoption. Recommended reading order: videos first for orientation, articles for context, then research for depth.
